ionbot

are you sick of never getting the library study hall spot you want?
wanted to attend biotechnology club but it was always full?
download ionbot, and run it either the night before or the lunch of an 8th period day.

how to use custom id

on ion, find the activity you want a spot in but is full. in the right hand bar,
search for a identifier called 'Activity ID'. that number is what you should input
into the 'Custom' section on ionbot. leave the 'Activity' selection blank.


how it was made

ionbot was built using selenium with python. i was annoyed because i could never get library
study halls, so i wanted to automate the process. i have a feeling this would not be approved
by the computer systems lab, but it's ok. shhhh.
